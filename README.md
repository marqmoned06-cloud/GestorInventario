# Gestor de Inventario de Tienda de Videojuegos 
 
## Descripción 
Aplicación de consola en Java que gestiona el inventario de una tienda de videojuegos. Permite aĄadir, buscar y vender artículos como videojuegos y consolas, con control de stock y excepciones personalizadas. 
 
## Compilación y ejecución 
1. Abre el proyecto en NetBeans 
2. Haz clic en Run o pulsa F6
